# JannenTalousHallinta
Henkilökohtainen taloudenhallintasovellus tulojen, menojen, säästöjen ja sijoitusten seurantaan yhdessä paikassa. Web-pohjainen, ensisijaisesti tietokoneelle suunniteltu, mutta toimii myös mobiilissa. Tavoitteena selkeä kokonaiskuva taloudesta ja myöhemmin AI-pohjainen analyysi.
